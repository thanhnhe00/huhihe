# Story Reading Platform

This repository contains the complete source structure for a Story Reading Platform graduation project.

## Repository Structure

```
be/
```

- Empty Spring Boot backend.
- Generate all backend source code here.
- Do not create another backend project.

```
fe/
```

- Final frontend project.
- This frontend is considered complete.
- Do NOT redesign the UI.
- Do NOT replace the frontend.
- Backend must be compatible with this frontend.

```
Project Report (.docx)
```

- This document is the primary source of requirements.
- Read it completely before generating any code.

---

## Development Priority

1. Project Report
2. Frontend
3. Existing repository structure

---

## Goal

Generate a complete Spring Boot backend that satisfies all requirements in the report and works directly with the existing frontend.

---

## Important Rules

- Keep the frontend unchanged whenever possible.
- Generate backend only inside the `be` folder.
- Follow the report before making assumptions.
- Build incrementally instead of generating everything at once.
